# Modern Text Workspace & Input Monitor

A clean, responsive, and modern text editing interface built in Python using `tkinter`. This application provides real-time document metrics, individual keypress event reflections, and localized file system operations within a sophisticated dark-themed layout.

## 🚀 Features

- **Modern Dark Theme:** Styled using integrated dark workstation palettes to reduce eye strain.
- **Live Text Metrics:** Real-time extraction of document geometry, tracking line counts, word length, and total characters.
- **Focused Input Stream:** Interactive status indicator showing the exact printable character or system control key executed during active focus.
- **Document Diagnostics:** Built-in analytics pop-up breaking down sentence structures and total word registry.
- **Safety Buffers:** Native undo configurations (`Ctrl + Z`) and confirmation warnings prior to wiping text states.

## 📋 Prerequisites

This application uses standard library modules and requires no external third-party installations. You only need Python installed on your system.

- Python 3.x
- `tkinter` (bundled natively with standard Python installers)
